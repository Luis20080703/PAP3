Serviço Indisponível

O servidor de backend não está ligado ou não é alcançável. Isto impede a aplicação de comunicar com a API.

Possíveis causas:

  • O servidor Laravel não foi iniciado (`php artisan serve`).
  • O servidor está a escutar noutro host/porto.
  • Firewall ou problema de rede local bloqueia a ligação.

Passos rápidos:

  1. Abra o terminal na pasta `Api` e execute `php artisan serve --host=127.0.0.1 --port=8000`.
  2. Verifique os logs em `Api/storage/logs/laravel.log`.
  3. Atualize a página quando o servidor estiver activo.
